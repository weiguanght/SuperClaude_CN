# SuperClaude 智能体 (Agents)

本目录包含针对各类专业 AI 智能体的定义文件。

## 可用智能体

- **deep-research.md** - 自主 Web 研究智能体
- **repo-index.md** - 代码仓库索引智能体
- **self-review.md** - 代码审查与质量检查智能体
- (更多智能体详见本目录下各文件)

## 重要提示

这些智能体定义是从 `plugins/superclaude/agents/` 复制过来的，用于包分发。

在更新智能体时：
1. 编辑 `plugins/superclaude/agents/` 中的文件。
2. 将更改同步复制到 `src/superclaude/agents/`。
3. 必须保持两个位置的同步。

在 v5.0 版本中，插件系统将直接使用 `plugins/` 目录。
