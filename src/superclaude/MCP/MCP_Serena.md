# Serena MCP 服务器 (Serena MCP Server)

**用途**：提供具备项目记忆和会话持久化能力的语义化代码理解。

## 触发场景
- 符号操作：重命名 (Rename)、提取 (Extract)、移动函数或类等。
- 全项目范围的代码导航与探索。
- 需要 LSP 集成的多语言项目。
- 会话生命周期管理：使用 `/sc:load`, `/sc:save` 或激活项目。
- 记忆驱动型开发工作流。
- 大型代码库分析（多于 50 个文件、架构复杂）。

## 何时选择
- **优于 Morphllm**：执行符号层面的操作，而非基于模式的编辑。
- **针对语义理解**：符号引用、依赖追踪、LSP 集成。
- **针对会话持久化**：项目上下文、记忆管理、跨会话学习。
- **针对大型项目**：需要架构级理解的多语言代码库。
- **不适用于简单编辑**：基础文本替换、风格强制执行、常规批量操作。

## 最佳协同组合
- **Morphllm**：由 Serena 分析语义上下文 → 由 Morphllm 执行精确编辑。
- **Sequential (顺序推理)**：由 Serena 提供项目上下文 → 由 Sequential 执行架构分析。

## 示例
```
"在所有地方重命名 getUserData 函数" → Serena (具备依赖追踪的符号操作)
"查找对此类的所有引用" → Serena (语义搜索与导航)
"加载我的项目上下文" → Serena (运行 /sc:load 激活项目)
"保存当前工作会话" → Serena (运行 /sc:save 持久化记忆)
"将所有 console.log 更新为 logger" → Morphllm (基于模式的替换)
"创建一个登录表单" → Magic (UI 组件生成)
```
