# SuperClaude 文档中心

## 🎯 核心认知

**SuperClaude 是 Claude Code 的一个上下文框架** —— 它通过安装行为指令文件，被 Claude Code 读取并以此增强其能力。

### 运行机制
1. **安装**：Python CLI 将上下文文件安装到 `~/.claude/` 目录。
2. **命令**：输入 `/sc:analyze` → Claude Code 读取 `analyze.md` 指令文件。
3. **行为**：Claude 采纳上下文文件中定义的行为模式。
4. **结果**：通过上下文切换实现增强的开发工作流。

## 🚀 快速入门

**新用户**：请参阅 [快速入门指南 →](Getting-Started/quick-start.md)
```bash
# 推荐 Linux/macOS 用户使用：
pipx install SuperClaude && SuperClaude install

# 或传统安装方式：
pip install SuperClaude && SuperClaude install

# 之后尝试：在 Claude Code 中输入 /sc:brainstorm "Web 应用创意"
```

## 🔑 核心概念

### 安装内容
- **Python CLI 工具**：管理框架的安装。
- **上下文文件**：位于 `~/.claude/` 下的 `.md` 行为指令。
- **MCP 配置**：可选的外部工具设置。

### 框架组成
- **21 个命令** (`/sc:*`)：工作流自动化模式。
- **14 位智能体** (`@agent-*`)：领域专家。
- **5 种模式**：行为修改模式。
- **6 个 MCP 服务器**：可选的外部扩展。

## 📊 框架与软件的对比

| 组件 | 类型 | 运行位置 | 功能 |
|-----------|------|---------------|--------------|
| **SuperClaude 框架** | 上下文文件 | 由 Claude Code 读取 | 修改 AI 的行为模式 |
| **Claude Code** | 软件 | 您的计算机 | 执行所有操作的核心 |
| **MCP 服务器** | 软件 | Node.js 进程 | 供 AI 调用的各类工具 |
| **Python CLI** | 软件 | Python 运行时 | 管理框架的安装 |

---

*请记住：SuperClaude 通过上下文增强 Claude Code 的能力，它本身不独立运行，一切操作都在 Claude Code 内部完成。*
