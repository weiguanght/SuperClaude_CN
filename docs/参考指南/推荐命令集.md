# 推荐命令集 (Suggested Commands)

## 安装与升级
```bash
# 推荐使用 pipx
pipx install SuperClaude
pipx upgrade SuperClaude
SuperClaude install

# 或使用标准 pip
pip install --upgrade SuperClaude
SuperClaude install

# 查看所有组件
SuperClaude install --list-components
```

## 开发环境搭建
```bash
# 创建虚拟环境 (推荐)
python3 -m venv .venv
source .venv/bin/activate  # Linux/macOS
# .venv\Scripts\activate   # Windows

# 安装开发/测试依赖
pip install -e ".[dev]"
```

## 测试与质量检查
```bash
# 运行 pytest
pytest -v
pytest --cov=superclaude --cov-report=html

# 代码格式化与风格检查
black .
mypy superclaude setup
flake8 superclaude setup
```

## 包构建与发布
```bash
# 清理并构建
rm -rf dist/ build/
python -m build

# 地安装测试
pip install -e .
```

## SuperClaude 常用功能
```bash
# 获取帮助
/sc:help

# 会话管理
/sc:load    # 恢复会话上下文
/sc:save    # 保存当前状态

# 常用开发命令
/sc:implement "功能描述"
/sc:test
/sc:analyze @file.py
/sc:research "主题"

# 委派专家
@agent-backend "设计 API"
@agent-security "审核鉴权逻辑"
```

---

*这些命令涵盖了从框架安装到日常开发流程的核心操作。*
