# SuperClaude 示例手册 (Cookbook)

本手册汇集了 SuperClaude 的各类实用操作案例，由浅入深，涵盖多个技术领域。为了提升可读性，我们将其拆分为三个核心板块：

## 板块概览

### 1. [基础示例集合](./基础示例.md)
**核心命令与单智能体工作流**
- 提供开箱即用的“复制-粘贴”式命令。
- 涵盖基础开发任务与简单的故障排除。
- **适用对象**：初学者、追求快速执行任务的用户。

### 2. [高级工作流集合](./高级工作流.md)
**多智能体协作与复杂编排**
- 专家级协作模式与企业级项目流。
- 侧重于会话管理、持久化及复杂系统开发。
- **适用对象**：资深用户、负责复杂架构设计的架构师。

### 3. [集成模式集合](./集成模式.md)
**框架集成与跨工具协同**
- 针对 React, Node.js, Python 等框架的深度集成。
- 包含性能优化方案与跨工具的自动化策略。
- **适用对象**：专家级用户、性能优化工程师。

---

## 学习成长路线

- **第一阶段（基础）**：建立信心，掌握 `/sc:brainstorm` 等核心命令。
- **第二阶段（协作）**：学习 `@agent-*` 多智能体配合，管理复杂项目。
- **第三阶段（集成）**：深度对接各类技术栈，成为框架集成专家。
- **第四阶段（专家）**：设计自定义集成模式，参与 SuperClaude 框架的建设。

---

## 常用命令速查

- `/sc:brainstorm`：交互式需求挖掘。
- `/sc:analyze`：代码库库库深度分析。
- `/sc:implement`：遵循最佳实践的功能实现。
- `/sc:test`：全方位的测试与验证。
- `/sc:load` / `/sc:save`：会话状态的加载与保存。
