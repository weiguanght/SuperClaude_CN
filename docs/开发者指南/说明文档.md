## 核心理念：面向上下文的配置 (Context-Oriented Configuration)
SuperClaude 实质上是供 Claude Code 读取的一系列 `.md` 指令文件。它**不是**独立运行的可执行软件。

**请注意**：当您在文档中看到 `/sc:` 命令时，这些是您在 Claude Code 对话框中输入的**触发模式**，而不是在终端执行的 CLI 命令。

### 扩展 SuperClaude 的步骤
1. **添加命令**：在 `~/.claude/commands/` 中创建新的 `.md` 文件。
2. **添加智能体**：在 `~/.claude/agents/` 中创建新的 `.md` 文件。
3. **添加模式**：在 `~/.claude/modes/` 中创建新的 `.md` 文件。

无需编译，无需部署——只要添加文件，Claude Code 就会在下次对话时自动读取它们。
