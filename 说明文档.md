## 🎯 **概述**

SuperClaude 是一个**元编程配置框架**，通过行为指令注入和组件编排，将 Claude Code 转换为结构化开发平台。它提供系统化的工作流自动化，配备强大的工具和智能代理。

## ⚡ **快速安装**

> **重要提示**：旧文档中提到的 TypeScript 插件系统尚未发布（计划在 v5.0 中推出）。关于当前的 v4.x 版本，请参考以下安装说明。

### **当前稳定版本 (v4.1.9)**

SuperClaude 目前使用斜杠命令。

**选项 1：使用 pipx (推荐)**
```bash
# 从 PyPI 安装
pipx install superclaude

# 安装命令 (安装所有 30 个斜杠命令)
superclaude install

# 安装 MCP 服务器 (可选，以增强功能)
superclaude mcp --list         # 列出可用的 MCP 服务器
superclaude mcp                # 交互式安装
superclaude mcp --servers tavily --servers context7  # 安装特定服务器

# 验证安装
superclaude install --list
superclaude doctor
```

安装完成后，重启 Claude Code 即可使用 30 个命令，包括：
- `/sc:research` - 深度网络研究 (通过 Tavily MCP 增强)
- `/sc:brainstorm` - 结构化头脑风暴
- `/sc:implement` - 代码实现
- `/sc:test` - 测试工作流
- `/sc:pm` - 项目管理
- `/sc` - 显示所有 30 个可用命令

**选项 2：直接从 Git 安装**
```bash
# 克隆仓库
git clone https://github.com/SuperClaude-Org/SuperClaude_Framework.git
cd SuperClaude_Framework

# 运行安装脚本
./install.sh
```

### **v5.0 即将推出 (开发中)**

我们正在积极开发全新的 TypeScript 插件系统 (详情见 issue [#419](https://github.com/SuperClaude-Org/SuperClaude_Framework/issues/419))。发布后，安装将简化为：

```bash
# 此功能尚未可用
/plugin marketplace add SuperClaude-Org/superclaude-plugin-marketplace
/plugin install superclaude
```

**状态**: 开发中。尚未设定 ETA。

### **增强性能 (可选 MCP)**

为了获得 **2-3 倍** 的执行速度并减少 **30-50%** 的 Token 消耗，您可以选择安装 MCP 服务器：

```bash
# 用于增强性能的可选 MCP 服务器 (通过 airis-mcp-gateway)：
# - Serena: 代码理解 (快 2-3 倍)
# - Sequential: Token 高效推理 (减少 30-50% Token)
# - Tavily: 用于深度研究的网络搜索
# - Context7: 官方文档查询
# - Mindbase: 跨所有对话的语义搜索 (可选增强)

# 注意：通过内置的 ReflexionMemory 提供错误学习 (无需额外安装)
# Mindbase 提供语义搜索增强 (需要 "recommended" 配置文件)
# 安装 MCP 服务器：https://github.com/agiletec-inc/airis-mcp-gateway
# 详见 docs/mcp/mcp-integration-policy.md
```

**性能对比：**
- **不使用 MCP**: 功能完备，标准性能 ✅
- **使用 MCP**: 快 2-3 倍，减少 30-50% Token ⚡

### **与 DR 代理架构对齐的自主 Web 研究**

SuperClaude v4.2 引入了全面的深度研究能力，实现了自主、自适应且智能的 Web 研究。

<table>
<tr>
<td width="50%">

### 🎯 **自适应规划**
**三种智能策略：**
- **仅规划 (Planning-Only)**: 针对明确查询直接执行
- **意图规划 (Intent-Planning)**: 针对模糊请求进行澄清
- **统一模式 (Unified)**: 协作完善计划 (默认)

### **研究命令使用**

```bash
# 自动深度的基础研究
/research "2024年最新 AI 进展"

# 受控的研究深度 (通过 TypeScript 选项)
/research "量子计算突破"  # 深度: exhaustive

# 选择特定策略
/research "市场分析"  # 策略: planning-only

# 领域过滤研究 (集成 Tavily MCP)
/research "React 模式"  # 领域: reactjs.org, github.com
```

### **研究深度级别**

| 深度 | 来源 | 跳数 | 时间 | 适用场景 |
|:-----:|:-------:|:----:|:----:|----------|
| **快速 (Quick)** | 5-10 | 1 | ~2min | 快速事实核查，简单查询 |
| **标准 (Standard)** | 10-20 | 3 | ~5min | 常规研究 (默认) |
| **深度 (Deep)** | 20-40 | 4 | ~8min | 全面的专题分析 |
| **详尽 (Exhaustive)** | 40+ | 5 | ~10min | 学术级深度研究 |

### **集成工具编排**

深度研究系统智能协调多个工具：
- **Tavily MCP**: 主要 Web 搜索与发现
- **Playwright MCP**: 复杂的网页内容提取
- **Sequential MCP**: 多步推理与综合
- **Serena MCP**: 记忆与学习持久化
- **Context7 MCP**: 技术文档查询
