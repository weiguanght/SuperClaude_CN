## 🎯 **工作原理**

<div align="center">

### **框架架构流程**

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│    用户输入      │────>│   Claude Code    │────>│   上下文文件    │
│  /sc:command    │     │    读取上下文     │     │(.md 行为定义)   │
└─────────────────┘     └──────────────────┘     └─────────────────┘
                                │                          │
                                ▼                          ▼
┌─────────────────┐      ┌──────────────────┐     ┌─────────────────┐
│    增强后的     │<─────│    行为激活      │<────│   MCP 服务器    │
│    响应         │      │                  │     │ (如果已配置)    │
└─────────────────┘      └──────────────────┘     └─────────────────┘
```

**奥秘所在**：当你输入 `/sc:brainstorm` 时，Claude 会读取已安装的 `.md` 文件中的行为指令，并以增强后的能力进行响应。

<tr>
<th width="50%">📦 第一步：安装 (终端执行)</th>
<th width="50%">💬 第二步：使用 (Claude Code 交互)</th>
</tr>
<tr>
<td valign="top">

```bash
# 使用 pipx 快速安装
pipx install SuperClaude && SuperClaude install

# 或使用传统的 pip
pip install SuperClaude && SuperClaude install

# 或通过 npm
npm install -g @bifrost_inc/superclaude && superclaude install
```

</td>
<td valign="top">

```text
# 交互式需求探索
/sc:brainstorm "任务管理类的 Web 应用"

# 分析现有代码
/sc:analyze src/

# 生成代码实现
/sc:implement "用户身份验证功能"

# 激活专业代理
@agent-security "审查身份验证流程"
```

</td>
</tr>
</table>

<details>
<summary><b>🎥 背后发生了什么？</b></summary>

1. **上下文加载**：Claude Code 通过 `CLAUDE.md` 导入行为定义的 `.md` 文件。
2. **模式识别**：识别 `/sc:` 和 `@agent-` 等触发模式。
3. **行为激活**：应用上下文文件中对应的指令。
4. **MCP 集成**：在可用时使用已配置的外挂工具。
5. **响应增强**：遵循框架模式提供全面、深入的响应。

</details>

## 🎯 **适用场景**

<div align="center">

### **SuperClaude vs 标准 Claude Code**

<table>
<tr>
<th width="50%">✅ 推荐使用 SuperClaude</th>
<th width="50%">💭 适合标准 Claude Code</th>
</tr>
<tr>
<td valign="top">

**完美契合：**
- 🏗️ 构建完整的软件项目
- 📊 带有质量把控的系统化工作流
- 🔄 复杂的多组件系统工程
- 💾 需要持久化状态的长期项目
- 👥 遵循统一标准的团队协作
- 🎯 对特定领域专家知识有需求

**典型例子：**
- “从零构建一个全栈应用”
- “实现安全的身份验证系统”
- “重构老旧的代码库”
- “创建全面的自动化测试套件”

</td>
<td valign="top">

**更适合于：**
- 💡 简单的问题咨询或概念解释
- ⚡ 一次性的代码小任务
- 📚 学习特定编程知识点
- 🧪 快速的原型验证或小实验
- 🔍 生成独立的代码片段
- ❓ 通用的编程辅助

**典型例子：**
- “解释 async/await 的工作原理”
- “写一个快速排序函数”
- “帮我 debug 这条错误消息”
- “把这个循环改写成函数式风格”

</td>
