## 推荐学习路径

### 1. 新生起步（首周基础）
- **目标**：建立信心，独立完成第一个开发任务。
- **路径**：`快速入门` -> `基础示例` -> `核心问题快速手册`。

### 2. 中坚力量（第 2-3 周）
- **目标**：掌握多智能体协作与技术深度。
- **路径**：`高级模式` -> `高级工作流` -> `MCP 服务器配置`。

### 3. 专家之路（进阶精通）
- **目标**：全面掌控框架，实现复杂系统的深度集成与诊断。
- **路径**：`集成模式` -> `故障排除` -> `诊断参考`。

## 常用命令与标志位速查

- **核心命令**：`/sc:load` (恢复会话), `/sc:save` (保存状态), `/sc:analyze` (分析), `/sc:test` (验证)。
- **深度控制**：`--think` (标准分析), `--think-hard` (深度), `--ultrathink` (竭尽分析)。
- **效率提升**：`--uc` (Token 压缩模式), `--orchestrate` (智能规划模式)。
- **工具激活**：`--c7` (官方文档), `--magic` (UI 设计), `--seq` (逻辑推导)。

## 开发框架快速入门

### React / Next.js
使用 `--c7 --magic` 配合 TypeScript 进行现代化的前端开发与组件设计。

### Node.js / Express
使用 `--c7 --seq` 进行遵循最佳实践的 API 设计与数据库集成。

### 全栈开发
使用 `--task-manage --all-mcp` 统筹整体应用工作流，并利用 Playwright 进行端到端测试。
