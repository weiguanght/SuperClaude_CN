# SuperClaude 命令参考指南 (Commands Reference)

此处列出了 SuperClaude 框架 v4.1.9+ 中可用的全部 30 个斜杠命令 (Slash Commands)。

## 命令分类

### 🧠 规划与设计 (Planning & Design)
- **`/brainstorm`** —— 提供多个视角的结构化头脑风暴会议。
- **`/design`** —— 系统设计与架构规划。
- **`/estimate`** —— 任务的工作量与时间估算。
- **`/spec-panel`** —— 多专家参与的技术规范分析。

### 💻 开发实施 (Development)
- **`/implement`** —— 代码实现工作流。
- **`/build`** —— 构建与编译工作流。
- **`/improve`** —— 提供代码改进建议。
- **`/cleanup`** —— 代码清理与重构。
- **`/explain`** —— 代码解释与文档说明。

### 🧪 测试与质量 (Testing & Quality)
- **`/test`** —— 测试工作流与测试用例生成。
- **`/analyze`** —— 代码与架构分析。
- **`/troubleshoot`** —— 调试与疑难解答。
- **`/reflect`** —— 反思与回顾。

### 📚 文档处理 (Documentation)
- **`/document`** —— 自动生成文档。
- **`/help`** —— 命令帮助及使用信息。

### 🔧 版本控制 (Version Control)
- **`/git`** —— Git 操作与工作流。

### 📊 项目管理 (Project Management)
- **`/pm`** —— 项目管理工作流。
- **`/task`** —— 任务管理与跟踪。
- **`/workflow`** —— 自定义工作流自动化。

### 🔍 调研与分析 (Research & Analysis)
- **`/research`** —— 支持并行搜索的深度网页调研。
- **`/business-panel`** —— 多专家参与的业务分析。

### 🎯 实用工具 (Utilities)
- **`/agent`** —— 专业 AI 智能体入口。
- **`/index-repo`** —— 仓库索引，用于优化上下文。
- **`/index`** —— `/index-repo` 的别名。
- **`/recommend`** —— 命令推荐。
- **`/select-tool`** —— 工具选择指导。
- **`/spawn`** —— 启动并行任务。
- **`/load`** —— 加载已保存的会话。
- **`/save`** —— 保存当前会话。
- **`/sc`** —— 显示所有可用的 SuperClaude 命令。

## 使用方法

所有命令均可通过 `/sc:` 命名空间前缀访问：

```bash
# 使用示例
/sc:brainstorm "如何改进用户身份验证"
/sc:implement "添加 JWT 身份验证"
/sc:test "为身份验证模块生成测试"
/sc:research "最新的安全最佳实践"
```

## 安装与更新

```bash
# 安装全部 30 条命令
superclaude install

# 列出已安装的命令
superclaude install --list

# 更新到最新版本
superclaude update
```

## 命令帮助

使用 `/sc:help` 获取任何命令的详细信息，或直接输入 `/sc` 查看所有可用命令。
