# 建议命令集 (Suggested Commands)

## 安装与设置 (Installation & Setup)
```bash
# 推荐的安装方法
pipx install SuperClaude
pipx upgrade SuperClaude
SuperClaude install

# 或者使用 pip
pip install SuperClaude
pip install --upgrade SuperClaude
SuperClaude install

# 查看组件列表
SuperClaude install --list-components

# 安装特定组件
SuperClaude install --components core
SuperClaude install --components mcp --force
```

## 开发环境配置 (Development Setup)
```bash
# 创建虚拟环境（推荐）
python3 -m venv .venv
source .venv/bin/activate  # Linux/macOS
# 或者
.venv\Scripts\activate     # Windows

# 安装开发专用依赖
pip install -e ".[dev]"

# 仅安装测试专用依赖
pip install -e ".[test]"
```

## 测试运行 (Testing)
```bash
# 运行所有测试
pytest

# 详细模式
pytest -v

# 带有覆盖率报告
pytest --cov=superclaude --cov=setup --cov-report=html

# 运行特定测试文件
pytest tests/test_installer.py

# 运行特定测试函数
pytest tests/test_installer.py::test_function_name

# 排除耗时较长的测试
pytest -m "not slow"

# 仅运行集成测试
pytest -m integration
```

## 代码质量检查 (Code Quality)
```bash
# 检查格式（不执行修改）
black --check .

# 应用格式化修改
black .

# 类型检查
mypy superclaude setup

# 代码风格检查 (Linter)
flake8 superclaude setup

# 执行全部质量检查
black . && mypy superclaude setup && flake8 superclaude setup && pytest
```

## 包构建 (Packaging)
```bash
# 清理构建环境
rm -rf dist/ build/ *.egg-info

# 构建包
python -m build

# 在本地安装以进行测试
pip install -e .

# 发布到 PyPI (仅限维护者)
python -m twine upload dist/*
```

## Git 操作 (Version Control)
```bash
# 确认状态（必做）
git status
git branch

# 创建功能分支
git checkout -b feature/your-feature-name

# 提交更改
git add .
git diff --staged  # 提交前确认
git commit -m "feat: add new feature"

# 推送更改
git push origin feature/your-feature-name
```

## macOS (Darwin) 特定命令
```bash
# 文件查找
find . -name "*.py" -type f

# 内容搜索
grep -r "pattern" ./

# 列出目录
ls -la

# 检查符号链接
ls -lh ~/.claude

# 默认使用 Python3
python3 --version
pip3 --version
```

## SuperClaude 使用示例
```bash
# 显示命令列表
/sc:help

# 会话管理
/sc:load    # 恢复会话
/sc:save    # 保存会话

# 常用开发命令
/sc:implement "功能描述"
/sc:test
/sc:analyze @文件名.py
/sc:research "调研主题"

# 智能体协同
@agent-backend "创建 API 端点"
@agent-security "审查身份验证逻辑"
```
