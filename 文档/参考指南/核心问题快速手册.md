# SuperClaude 常见问题快速手册 (Common Issues) 🚀

针对最频繁出现的问题提供实用解决方案。

## 五大快速修复方案 (解决 90% 的问题)

### 1. 命令在 Claude Code 中不工作 ⚡
- **现象**：输入 `/sc:brainstorm` 无响应。
- **对策**：**完全重启** Claude Code。

### 2. 安装核验
核验版本号 `python3 -m SuperClaude --version` 是否为 4.1.5。如果不是，请通过 `pipx upgrade` 或 `pip install --upgrade` 进行更新，并重新运行安装命令。

### 3. 权限问题
- **现象**：权限被拒绝或出现 PEP 668 错误。
- **对策**：推荐使用 `pipx` 安装；或使用 `pip install --user`；或通过 `chown` 修复 `~/.claude` 目录权限。

### 4. MCP 服务器问题
如果怀疑 MCP 导致卡顿，可附加 `--no-mcp` 测试純上下文模式；同时核验 Node.js 版本 (16+)。

### 5. 组件缺失
运行 `SuperClaude install --components core commands agents modes --force` 以强制补充所有必需组件。

---

## 平台特定建议

- **Windows**: 确保手动设置环境变量 `%USERPROFILE%\.claude` 并指定安装目录。
- **macOS**: 推荐通过 `brew` 安装 Python3 和 Node，然后再进行 pip 安装。
- **Linux**: 确保安装了 `python3-pip` 和 `nodejs`。

---
